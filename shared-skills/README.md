# Shared Skills

Claude skills for the Backlinko/Semrush content team. These can be uploaded to Claude web apps or used with Claude Code.

## How to Use (Claude Web App)

1. Download the skill folder you want (e.g., `thesis/`)
2. Zip the folder
3. In Claude, go to **Settings > Capabilities**
4. Enable "Code execution and file creation"
5. Under **Skills**, click "Upload skill" and select your zip file

The skill will auto-trigger when Claude recognizes a relevant task.

## Contributing

To add or update a skill, create a PR with:
- A folder named after the skill
- A `Skill.md` file with the skill logic
- A `README.md` explaining what it does and when to use it
